# v1.3.5 (2026-02-19)

## 逻辑修复：底导 Tab 数据隔离

### 问题
- 同一个条目会同时出现在首页、灵感页和选题页
- 数据冗余，逻辑混乱

### 解决方案

#### 1. 首页 (HomeScreen) - 纯粹捕获界面
- **移除所有数据列表**
- 只显示 Logo + 捕获按钮
- 回归极简设计

#### 2. 灵感页 (IdeasScreen) - 收件箱 (Stage 0)
- 只显示初始灵感状态 (`status == idea`)
- 当灵感被"开始策划"后，立即从本页消失

#### 3. 选题页 (ProjectsScreen) - 项目管理区 (Stage 1+)
- 只显示已策划的条目 (`status != idea`)
- 包括：策划中、制作中、已完成

### 数据流
```
CaptureScreen → 灵感列表(Idea) → 开始策划 → 选题列表(Planning/InProgress/Completed)
```

### 流程优化
- 灵感即项目：点击"开始策划"后状态变为 planning
- 数据单向流动：一个想法在同一时间只存在于一个列表

## UI 升级：Zen Pod Premium 极简首页

### 设计理念
- 极简悬浮舱 (Zen Pod) 设计
- 高级感、空气感、通透感

### Layer 0: 空气渐变背景 (底层)
- 全屏 Container
- LinearGradient: `Alignment.topCenter` → `Alignment.bottomCenter`
- 颜色: `#F8FAFC` (极淡冷灰白) → white (纯白)

### Layer 1: 能量光晕 (中层) - 极致物理模糊
- 使用 `ImageFilter.blur` 实现无边界的极光弥散场
- 解除边界裁剪 (`clipBehavior: Clip.none`)
- 发光核心：200×200 圆形，40% 透明度品牌青 (#0891B2)
- 极致模糊：sigmaX: 100, sigmaY: 100
- 定位：`Alignment(0.0, -0.1)` (中心偏上)
- 效果：没有任何物理边缘的、如烟雾般化开的青色气场

### Layer 2: 捕获大按钮 (FAB)
- 尺寸: 72×72 圆形
- 背景: 纯白
- 阴影: 品牌青 15% opacity, blur 24, offset (0,8)
- 图标: `Icons.add`, 36px, 品牌青色 (#0891B2)
- 触感反馈: 按下时缩放 + 阴影加深 + 触觉震动
