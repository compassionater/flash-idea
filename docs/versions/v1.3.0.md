# v1.3.0 (2026-02-19)

## 重大重构 - 统一内容生命周期

### 状态系统重构
将原有的双布尔值（`isProject`、`isShooting`）替换为单一的 `IdeaStatus` 枚举，实现线性进度追踪：

| 状态 | 值 | 说明 |
|------|-----|------|
| idea | 0 | 灵感 - 默认状态 |
| planning | 1 | 策划中 |
| inProgress | 2 | 制作中 |
| completed | 3 | 已完成 |

### 优势
- 消除状态歧义：一个字段明确表示进度
- 线性流程：灵感 → 策划中 → 制作中 → 已完成
- 易于扩展：未来可轻松添加中间状态

### UI更新
- **灵感卡片**：显示当前状态标签（策划中/制作中/已完成）
- **详情页**：新增状态进度条（Stepper风格）
- **状态更新**：支持在详情页直接切换状态

### 技术变更
- `Idea` 模型新增 `IdeaStatus status` 字段
- 移除 `bool isProject` 字段
- 移除 `bool isShooting` 字段
- Hive 生成新的 `IdeaStatusAdapter`
