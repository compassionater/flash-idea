# v1.4.0 (2026-02-19)

## 核心重构：数据隔离与状态管理

### 统一状态系统
- 引入 `IdeaStatus` 枚举：灵感 → 策划中 → 制作中 → 已完成
- 数据迁移：兼容旧版本 `isProject` 布尔值

### 底导 Tab 数据隔离
- **首页**：纯粹捕获界面，只显示 Logo + 捕获按钮
- **灵感页**：仅显示初始灵感状态，转换为选题后自动消失
- **选题页**：仅显示已策划的条目（策划中/制作中/已完成）

---

## UI 升级：极光弥散与毛玻璃

### 首页 Zen Pod 设计
- **极光弥散场**：ImageFiltered 高斯模糊（sigma 100+）打造突破边界的青色气场
- **呼吸动画**：3秒周期的光晕脉动效果
- **FAB 交互**：按压缩放 + 触觉反馈（HapticFeedback）
- **品牌水印**：极简文字标识
- **漂浮记忆碎片**：背景动画记忆气泡，暗示"活的思维池"

### 输入页 Glassmorphism
- **磨砂水晶画布**：ClipRRect + BackdropFilter 构建毛玻璃卡片
- **情感化文案**："又有新想法了！"
- **发酵动画**：点击发送后隐藏输入框，显示 "💡 灵感已捕获，正在静静发酵..." 停留1.5秒后退出

### 底部导航栏
- **毛玻璃效果**：BackdropFilter 模糊 + 半透明背景
- **弹簧动画**：选中图标弹性缩放
- **切页过渡**：AnimatedSwitcher 淡入淡出

---

## 视觉优化

### 色彩系统统一
- 品牌色标准化：#0891B2 (Cyan)
- 语义色常量：状态色、分类色等
- 清除硬编码颜色

### 卡片设计
- **双层柔和阴影**：近影 + 远影，营造悬浮纸片效果
- **分类色彩指示条**：卡片左侧 4px 彩色竖条，颜色跟随分类
- **分类标签着色**：使用对应分类颜色，不再统一灰色
- 灵感卡片与选题卡片样式统一

### 列表动效
- **交错入场**：列表项依次滑入动画
- **呼吸空状态**：无内容时显示呼吸图标

### 详情页
- **状态进度条**：发光圆点 + 连接线动画
- 状态更新弹窗

---

## 技术改进

### 性能优化
- **RepaintBoundary**：应用于静态但有动画的组件（气泡、光晕），利用光栅缓存
- **模糊优化**：背景光晕缓存渲染，避免每帧重新模糊

### 代码优化
- 移除未使用/冗余组件层
- 数据模型简化

## 功能补全

### 无压分类
- **移除捕获页分类选择器**：灵感默认保存为"未分类"
- **快速标签**：`IdeaCard` 分类指示器可点击，一键打开底部弹窗选择分类
