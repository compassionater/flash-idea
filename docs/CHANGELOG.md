# 更新日志

## v1.4.0 (2026-02-19)
[详细文档](./docs/versions/v1.4.0.md)

### 核心重构：数据隔离与状态管理
- **统一状态系统**：IdeaStatus 枚举（灵感/策划中/制作中/已完成），兼容旧版本数据迁移
- **底导 Tab 数据隔离**：首页/灵感页/选题页数据分流，灵感转为选题后自动从灵感列表移除

### UI 升级：极光弥散与毛玻璃
- **首页 Zen Pod**：极光弥散场（ImageFiltered 高斯模糊）+ 呼吸动画 + FAB 按压反馈 + 品牌水印 + 漂浮记忆气泡
- **输入页 Glassmorphism**：毛玻璃卡片 + 发酵动画 + 情感化文案
- **底部导航栏**：毛玻璃效果 + 弹簧动画 + 淡入淡出切页

### 视觉优化
- **色彩系统统一**：品牌色 #0891B2 + 语义色常量
- **卡片设计**：双层柔和阴影 + 分类色彩指示条 + 分类标签着色
- **列表动效**：交错入场动画 + 呼吸空状态
- **详情页**：状态进度条发光动画

### 功能补全
- **无压分类**：移除捕获页分类选择器，默认"未分类"，卡片上一键快速分类

### 技术改进
- **性能优化**：RepaintBoundary 缓存 + 模糊渲染优化
- **代码清理**：移除冗余组件层
